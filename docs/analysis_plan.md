# Plano de Análise

## 1. Objetivo do plano

Este documento define a sequência de análise do projeto Business Performance Dashboard — Power BI.

O objetivo é garantir que a construção do dashboard siga uma lógica de negócio clara, evitando que o projeto se torne apenas uma coleção de gráficos sem diagnóstico.
